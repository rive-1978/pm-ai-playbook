## Learning Loop

**Before starting a new task,** review existing rules and hypotheses for this domain. 
**Apply rules by default.** Check if any hypotheses can be tested with today's work.

**At the end of each task,** extract insights.
Store them in domain folders, e.g.:
    /06-system/learnings/pm-os/
        `knowledge.md` (facts and patterns)
        `hypotheses.md` (need more data)
        `rules.md` (confirmed - apply by default)

**Maintain** `06-system/learnings/INDEX.md` that routes to each domain folder. 

When a hypothesis gets confirmed 5+ times, 
promote it to a rule
When a rule is contradicted by new data,
demote it to a hypothesis.
